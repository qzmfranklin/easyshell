=========
easyshell
=========

Library for creating recursive shells.


Installation
============

From pip3::

    $ pip3 install -U easyshell

From source::

    $ ./setup.py -U install

Example Shell
=============

To experience this kind of shells, please try out the example shell that comes
with the source code::

    $ python3 -m easyshell
