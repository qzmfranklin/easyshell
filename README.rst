=========
easyshell
=========

Library for creating recursive shells.


Installation
============

From pip3::

    $ pip3 install --upgrade easyshell

From source::

    $ python3 setup.py --quiet install

Example
=======

You can try out the example shell that ships with the source code::

    $ ./my_shell.py
